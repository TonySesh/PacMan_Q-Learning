Описание: Классическая аркадная игра PacMan, в которой персонаж самостоятельно учится играть с помощью алгоритма Q-learning (обучение с подкреплением).
За основу был взял проект: https://github.com/usawa/pypacman
Далее он был переделан под обучение с подкреплением. 
Проект был сделан в образовательных целях.

Стек: Python, Pygame, Q-learning, NumPy, Jupyter Noteboook

Про Q-обучение: 
При первом запуске создается новая Q-таблица. 
Создается файл q_table_with_ghost.pkl

Файл themes - текстуры игры (некоторые рисовались вручную)

Как происходит обучение:
1)  Пакман смотрит на ситуацию
Он определяет, где точка, где призрак, куда можно идти. Находит в шпаргалке эту ситуацию и смотрит оценки.
2) Выбирает действие
Пакман не всегда выбирает самое лучшее действие — иногда он экспериментирует. 
3) Делает шаг и получает награду или наказание
Съел точку → +10 очков 
Приблизился к точке → маленький плюс 
Приблизился к призраку → минус 
Призрак съел Пакмана → -1000
4) Обновляет шпаргалку
